# VAE_Fashion-MNIST
**A simple VAE Model** with ~98.8% accuracy on Fashion MNIST Dataset

train.py is the code to train the model. Note that the accuracy might differ based on the test model's accuracy.

## Prerequisites

- Pytorch
- Torchvision
- Matplotlib
- tqdm
- Cuda
